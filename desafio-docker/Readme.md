# Desafio FullCycle

### 📋 Pré-requisitos

- Docker instalado localmente

### 🔧 Execução

Construa a imagem docker:
```
docker build -t desafio-fullcycle .
```
Execue a imagem docker:
```
docker run desafio-fullcycle
```
